# 平台货源标准字段知识库

## 用途
定义平台标准字段、中文名、校验规则、是否必填与示例值。

## 字段定义模板
| 字段编码 | 中文名 | 必填 | 类型 | 校验规则 | 示例值 |
|---|---|---|---|---|---|
| A04_origin_address_detail | 装货详细地址 | 是 | string | 需到街道门牌 | 成都市青羊区光华大道三段188号 |
| A08_destination_address_detail | 卸货详细地址 | 是 | string | 需到街道门牌 | 广州市白云区机场路1268号 |
| B03_goods_weight_ton | 货物重量（吨） | 是 | number | >0 | 18 |
| C01_vehicle_type | 车型要求 | 是 | enum | 冷藏车/保温车/厢式 | 冷藏车 |
| C02_vehicle_length | 车长要求 | 是 | enum | 4.2/6.8/7.6/9.6/13/17.5 | 9.6米 |
| D01_load_time_start | 最早装货时间 | 是 | datetime | 北京时间 | 2026-03-28 08:00 |
| E01_freight_price | 运费（元） | 是 | number | >=0 | 30000 |
| E03_payment_method | 付款方式 | 是 | enum | 到付/预付/月结/回单付 | 到付 |
| B06_temperature_requirement | 温度要求 | 冻品必填 | string | 冷链货建议必填 | -18℃以下 |
